Aula Prática 3 - Menu

Esta é uma página desenvolvida com o objetivo de ilustrar a construção e estilização de um menu horizontal em uma página HTML, corroborando para a execução do projeto final dos alunos

Além da estrutura básica de um documento HTML (Incluindo tags "HTML", "HEAD" e "BODY") Foram utilizadas, para elaboração do Menu, as tags "Nav", "ul", "li" e "a". A tag "Nav" Representa um conjunto de links para navegação, "ul" uma lista ordenada, a tag "li" um elemento, ou valor, em uma lista. E a tag "a" para carregar hiperlinks.
